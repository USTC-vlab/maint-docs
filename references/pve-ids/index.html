    Proxmox 虚拟机 ID 分配¶

    ID 或 ID 段 分配情况
    100-199 vlab 服务
    如 gateway, web 等
    200-799 未分配
    800-899 托管在 vlab 上的其他服务
    如 USTC OJ, Verilog OJ 等
    900-999 vlab 维护小组自己使用的虚拟机
    1000-? 用户创建的虚拟机
